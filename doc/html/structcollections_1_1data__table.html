procedure, public collections::data_table::append_column ( )

Appends a single column onto the end of the data_table.

Syntax
subroutine append_column(class(data_table) this, class(*) x(:), class(errors) err)
Parameters
[in,out]thisThe data_table object.
[in]xThe N-element array where N is equal to the number of rows in this data_table. A copy of each item is made, and the data_table takes care of management of the memory occupied by each copy.
[in,out]errAn optional errors-based object that if provided can be used to retrieve information relating to any errors encountered during execution. If not provided, a default implementation of the errors class is used internally to provide error handling. Possible errors and warning messages that may be encountered are as follows.
  • FCORE_ARRAY_SIZE_ERROR: Occurs if x does not have the same number of items as the data_table has rows.
  • FCORE_OUT_OF_MEMORY_ERROR: Occurs if there is insufficient memory.
