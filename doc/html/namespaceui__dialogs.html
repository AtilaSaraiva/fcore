Shows the user a message box dialog.

Parameters
[in]txtThe text to display in the message box.
[in]titleThe title to display in the header of the message box.
[in]buttonsDefines what button group to display. The following choices are acceptable: MSGBX_BTN_OK, MSGBX_BTN_OK_CANCEL, MSGBX_BTN_YES_NO, MSGBX_BTN_YES_NO_CANCEL, MSGBX_BTN_RETRY_CANCEL, MSGBX_BTN_ABORT_RETRY_IGNORE, MSGBX_BTN_CANCEL_RETRY_CONTINUE.
[in]iconDefines what icons to display. The following choices are acceptable: MSGBX_NO_ICON, MSGBX_NO_BUTTON, MSGBX_ICON_ERROR, MSGBX_ICON_WARNING, MSGBX_ICON_QUESTION, MSGBX_ICON_INFORMATION.
[in]parentAn optional argument defining a pointer to the parent window. If not supplied, no parent is assumed.
Returns
The results from the dialog.
